# Personal Portfolio Website

## Overview

This project is a personal portfolio website developed using HTML5 and CSS3. It is designed to present my profile, technical skills, projects, hobbies, and contact information through a clean and responsive interface.

## Features

- Responsive and user-friendly design
- Dedicated About Me section
- Skills showcase
- Projects portfolio section
- Hobbies and interests page
- Contact page for communication
- Consistent layout and styling across all pages

## Technologies Used

- HTML5
- CSS3

## Project Structure

```text
├── index.html
├── about.html
├── skills.html
├── project.html
├── hobbies.html
├── contact.html
├── style.css
└── image/
    └── image.png
```

## Pages

- **Home** – Landing page and introduction
- **About** – Personal background and information
- **Skills** – Technical skills and competencies
- **Projects** – Showcase of completed projects
- **Hobbies** – Personal interests and activities
- **Contact** – Contact details and communication section

## Purpose

The purpose of this portfolio website is to establish a professional online presence and demonstrate my web development skills, projects, and personal profile.
